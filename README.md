# 🧠 ECHO - Personal AI Memory System

ECHO is a sophisticated system that simulates how AI can develop long-term memory of a user using structured data over time. It captures activities, detects behavioral patterns, and generates meaningful insights.

## Features

- **Event Logging**: Capture activities with timestamps, types, durations, and tags
- **Temporal Analysis**: Detect peak hours, weekly patterns, and activity trends
- **Memory Engine**: Build a dynamic user profile that evolves over time
- **AI Reflections**: Generate personalized insights using pattern detection (optional OpenAI integration)
- **Conversational Interface**: Ask questions about your behavior naturally
- **Interactive Dashboard**: Streamlit UI for easy interaction
